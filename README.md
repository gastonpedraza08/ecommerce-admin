## Como usar

<!-- Example: -->
```bash
# Clona el repositorio
$ git clone https://github.com/gastonpedraza08/magazinca

# Instala las dependencias
$ cd client
$ yarn
$ cd ..
$ cd server
$ yarn
$ cd ..

# Corre la app
$ cd server
$ yarn start
$ cd ..
$ cd client
$ yarn start
```
